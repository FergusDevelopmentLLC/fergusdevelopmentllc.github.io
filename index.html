United States Federal Lands (GeoDev Web Stack)

This project involved building a full web stack from database to Vue.js front end. Static geojson, PostGIS, and MongoDB queries are compared to deliver intricate geospatial datasets. Web server based on Node.js and supporting libraries. For full details, see the github repo.

  • See the map: United States Federal Lands
  • Github: https://github.com/FergusDevelopmentLLC/geodev-node-rest
  • Data Source(s): Data.gov
  • Technologies/Platforms: PostgreSQL, PostGIS, MongoDB, REST API, GeoJSON, Node.js, NPM, Express.js, Knex.js, Mongoose.js, Joi.js, Bootstrap, Vue.js, Oracle VirtualBox

U.S. Billboard #1 hit singles 1970-1979

Choropleth map showing the U.S. Billboard #1 hit singles by artist origin city between 1970-1979. Python script was written to scrape web sources and create local datasets.

  • See the map: U.S. Billboard #1 hit singles 1970-1979
  • Data Sources: Ultimate Music Database, Wikipedia, U.S. Census, NOAA, Natural Earth
  • Technologies/Platforms: QGIS, Carto, PostGIS geospatial queries, Python

Rio Grande National Forest Subsurface Mineral Rights

Highlights areas within the Rio Grande National forest that have no subsurface mineral rights. This is a traditional static PDF map created in Arc Desktop, prepared for Rocky Mountain Wild.

Tecnologies/Platforms: ArcMap, GIS

  • See the map: Rio Grande National Forest Subsurface Mineral Rights
  • Data Source(s): Bureau of Land Management, Colorado Parks and Wildlife, United States Forest Service, Wilderness.net
